  - 显示协议链接：《个人信息授权书》《投保提示书》等
  - 显示“立即投保”按钮

## 2. 保障内容页面测试
- 2.1 验证保障额度展示
  - 保险期间内赔付限额：600万
  - 一般医疗保险金：300万
  - 120种特定疾病医疗保险金：600万
  - 特定技术医疗保险金：600万
- 2.2 验证“展开更多”功能
  - 点击可查看完整保障明细
- 2.3 验证底部导航
  - 我要投保、产品详情、理赔流程可切换

## 3. 投保人信息填写测试
- 3.1 验证姓名输入框
  - 可输入中英文，不能为空
- 3.2 验证“中国境内”选项
  - 是 / 否 切换
- 3.3 验证证件类型及号码
  - 默认身份证，校验格式
- 3.4 验证手机号及验证码
  - 手机号格式校验
  - 获取验证码按钮可点击
- 3.5 验证联系地址（省）
  - 下拉选择，不能为空

## 4. 被保人信息填写测试
- 4.1 验证“新增被保人”功能
  - 可新增本人/父母/配偶
- 4.2 验证被保人姓名
  - 必填，支持中英文
- 4.3 验证被保人“中国境内”选项
  - 是 / 否 切换
- 4.4 验证被保人证件号码
  - 身份证格式校验
- 4.5 验证有无医保
  - 有医保（含新农合） / 无医保
- 4.6 验证联系地址选项
  - 同投保人联系地址 / 其他地址
- 4.7 验证被保人联系地址（省）
  - 当选择“其他地址”时可编辑

## 5. 产品计划选择测试
- 5.1 验证计划选择
  - 计划一（1万免赔）
  - 计划二（5000免赔）
- 5.2 验证保障生效日期
  - 默认展示：2026-05-23
  - 可修改，不能早于投保当日

## 6. 协议确认与提交测试
- 6.1 验证协议勾选
  - 必须勾选所有协议才能提交
- 6.2 验证保费展示
  - 显示“-元/月”或动态计算后金额
- 6.3 验证“立即投保”按钮
  - 信息完整且协议勾选后可点击
  - 提交后跳转支付或确认页

## 7. 异常场景测试
- 7.1 必填项为空提示
- 7.2 身份证号格式错误提示
- 7.3 手机号格式错误提示
- 7.4 未勾选协议时提交提示
- 7.5 网络异常时提交提示
