# 平安e生保·百万医疗2026 测试用例

## 1. 页面与入口测试
- 1.1 验证“保障内容”可正常展开/收起
- 1.2 验证“产品详情”“理赔流程”可正常切换
- 1.3 验证“专属管家”入口功能
- 1.4 验证“立即投保”按钮可点击
- 1.5 验证“查看详情”链接跳转正确

## 2. 投保人信息模块
- 2.1 姓名输入框
  - 2.1.1 支持中英文输入
  - 2.1.2 不支持特殊字符
  - 2.1.3 为空时提交提示
- 2.2 中国境内选择
  - 2.2.1 选择“是”
  - 2.2.2 选择“否”
- 2.3 身份证号输入
  - 2.3.1 格式校验（18位）
  - 2.3.2 重复校验
- 2.4 手机号码
  - 2.4.1 11位数字校验
  - 2.4.2 发送验证码功能
- 2.5 手机验证码
  - 2.5.1 验证码有效期
  - 2.5.2 错误提示
- 2.6 联系地址（省）
  - 2.6.1 下拉选择
  - 2.6.2 必填校验

## 3. 被保人模块
- 3.1 新增被保人
  - 3.1.1 可新增多个被保人
  - 3.1.2 删除被保人功能
- 3.2 被保人类型
  - 3.2.1 本人
  - 3.2.2 父母
  - 3.2.3 配偶
- 3.3 被保人姓名
  - 3.3.1 输入校验
- 3.4 被保人身份证
  - 3.4.1 格式校验
- 3.5 有无医保
  - 3.5.1 有医保（含新农合）
  - 3.5.2 无医保
- 3.6 联系地址
  - 3.6.1 同投保人地址
  - 3.6.2 其他地址（省选择）

## 4. 产品计划选择
- 4.1 计划一（1万免赔）
  - 4.1.1 保费展示正确
  - 4.1.2 保障内容正确
- 4.2 计划二（5000免赔）
  - 4.2.1 保费展示正确
  - 4.2.2 保障内容正确
- 4.3 计划切换后保障内容联动更新

## 5. 保障生效日期
- 5.1 默认显示次日日期（2026-05-23）
- 5.2 不可选择过去日期
- 5.3 不可选择大于规定范围日期

## 6. 协议与授权
- 6.1 协议勾选
  - 6.1.1 未勾选时不可提交
  - 6.1.2 勾选后可提交
- 6.2 各协议可点击查看
  - 6.2.1 《个人信息授权书》
  - 6.2.2 《投保提示书》
  - 6.2.3 《责任免除》
  - 6.2.4 《投保须知》
  - 6.2.5 《特别告知》
  - 6.2.6 《保险条款》
  - 6.2.7 《产品说明书》
  - 6.2.8 《服务手册》
  - 6.2.9 《投保场景隐私政策》
  - 6.2.10 《扣款授权协议》

## 7. 保费与支付
- 7.1 保费按月展示（-元/月）
- 7.2 点击“立即投保”后跳转支付
- 7.3 支付方式选择（若有）

## 8. 异常与边界测试
- 8.1 必填项为空提交
- 8.2 身份证号重复
- 8.3 手机号未验证提交
- 8.4 网络异常时提交
- 8.5 重复提交同一保单

## 9. 兼容性测试
- 9.1 手机浏览器
- 9.2 微信内打开
- 9.3 iOS/Android 系统
